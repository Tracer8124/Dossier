# Tanvi Ligade - Machine Learning Engineering Portfolio

Welcome to the source code for my interactive portfolio. I am a Machine Learning Engineer with 3 years of experience in designing, developing, and deploying AI/ML solutions using Python.

## 🚀 About the Project

This portfolio is built using **Next.js** and **Framer Motion** to deliver a sleek, modern, and highly interactive presentation of my resume. It features dark mode aesthetics, smooth parallax scrolling, and interactive modules to explore my background in Machine Learning.

### Key Features
- **Cinematic 3D Parallax Hero**: Smooth scroll-based animations and a full-screen interactive background that tracks mouse physics. 
- **Interactive Project Modals**: Click to explore detailed task breakdowns and technology stacks for each ML project.
- **Experience Timeline**: Scroll-triggered animated timeline detailing my roles at Santander Bank, KPIT, and DRDO.
- **Skills Matrix**: Visual representation of my core proficiencies (Python, AWS, NLP/LLM, etc.) and Education.

## 🛠️ Built With

- **Framework:** [Next.js (App Router)](https://nextjs.org/)
- **UI & Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** TypeScript

## 💻 Getting Started

First, install the necessary dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to explore the portfolio locally.
